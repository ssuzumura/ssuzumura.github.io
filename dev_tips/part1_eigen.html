テキストによる説明

Eigenはヘッダーファイルとして配布されており，それらをインクルードするだけで使用することができます． 以下の手順に従い，サンプルコードが実行できることを確認して下さい．

  • こちらの公式サイトから最新版をダウンロードして解凍します (画面右端のGet Itからダウンロード)
  • 解凍後，フォルダ名をeigen33とします(33はバージョンを表す).
  • eigen33フォルダを下記のディレクトリ直下に移動させます.
    • Windowsの場合 C:/
    • Mac/Linuxの場合 /usr/local/include/
  • 以下のCMakeLists.txtファイルを作成してください. 
    cmake_minimum_required(VERSION 2.8)
add_definitions("-std=c++11 -Wall -O3 -march=native")
include_directories(”C:/eigen33”) # eigen33のパスを指定
add_executable(main test.cc)
  • 以下のtest.ccファイルを作成してください. 
    #include <iostream>
#include <Eigen/Core>

int main(void)
{
  Eigen::MatrixXd A(3,3);
  A << 1,2,3,
       4,5,6,
       7,8,9;

  std::cout << A << std::endl;
  return 0;
}
  • test.ccをコンパイルして実行すると
    　1 2 3
    　4 5 6
    　7 8 9
    が表示されます．コンパイル方法についてはCMakeに関する記事をご確認ください．
