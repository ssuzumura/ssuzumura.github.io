Eigenのインストール (全体の目次に戻る)

Eigenはヘッダーファイルとして配布されており，それらをインクルードするだけで使用することができます． 以下の手順通りに作業を進めると、サンプルコードを実行できます．

  • こちらの公式サイトから最新版をダウンロードして解凍します (画面右端のGet Itからダウンロード)
  • 解凍後，フォルダ名をeigen33とします(33はバージョンを表す).
  • 以下のディレクトリ構造でファイルを保存していきます．
    • eigen33 (eigenが保存されたフォルダ)
    • CMakeLists.txt (cmakeの設定ファイル)
    • test.cc (サンプルコード)
    CMakeLists.txt： 
    cmake_minimum_required(VERSION 2.8)
add_definitions("-std=c++11 -Wall -O3 -march=native")
include_directories("eigen33") # eigen33のパスを指定
add_executable(main test.cc)
    test.cc： 
    #include <iostream>
#include <Eigen/Core>

int main(void)
{
  Eigen::MatrixXd A(3,3);
  A << 1,2,3,
       4,5,6,
       7,8,9;

  std::cout << A << std::endl;
  return 0;
}
  • test.ccをコンパイルして実行すると
    　1 2 3
    　4 5 6
    　7 8 9
    が表示されます．コンパイル方法についてはCMakeに関する記事もご確認ください．
