C++11のコンパイル環境構築 (全体の目次に戻る)

本サイトではC++11と呼ばれるコンパイラを用いてプログラミングしていきます． ここではコンパイラ、およびコンパイルを自動化するCMakeをインストールします．

Linuxにおける作業

Linux の場合は Ubuntu OS でしたら以下を実行すれば環境構築は完了です． 
sudo apt-get install gcc cmake
ただし、バージョン 4.7 以降の GCC をインストールして下さい(g++ -v でバージョン確認)． GCC をソースからコンパイルする場合は例えばこちらをご参照ください． CentOSをご利用の方はこちらをご参照ください．

Windowsにおける作業

WindowsではMSYS2を用いてGCCをインストールするのが簡単です．
インストール手順：
  • こちらの公式サイトからmsys2-x86_64をダウンロードしてインストール（デフォルト値のままでインストール）
  • インストール後に実行されるシェル端末に「pacman -Sy pacman」と打ち込んでEnter
  • 一度端末を閉じて、スタートメニューから「MSYS2 Shell」を実行
  • 端末上に「pacman -Syu」と打ち込んでEnter
  • 再度端末を閉じて、スタートメニューから「MSYS2 Shell」を実行（端末が立ち上がらない場合は「C:\msys64\msys2_shell.cmdを実行」）
  • 端末上に「pacman -Su」と打ち込んでEnter
  • 端末上に「pacman -S gcc cmake make」と打ち込んでEnter
右クリックでMSYS2を実行する方法
この機能は設定しなくても問題ありません． もし設定する場合は、レジストリーの内容を書き換えるため、自己責任で行ってください．
こちらから「open_msys2.reg」というファイルをダウンロードして実行してください． 特定のフォルダを右クリックし「Open MSYS2 here」を押すとMSYS2のシェル端末を開くことができます．
